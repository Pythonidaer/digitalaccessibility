Screen Readers & Desktop Testing

Notes from the former Tools page: the “Desktop” subsection under “Platform-Specific Tools,” without the Microsoft 365 bullet (that bullet is on Office / Document Tools).

Desktop

  • Keyboard navigation, built-in accessibility tools.
  • Screen readers like:
    • NVDA free & open source
      • Allows you to slow down and speed up speech rate playback
    • JAWS commercial product (requires a license)
      • Has trial periods
      • Offers advanced features
      • More in-depth customization, scripting, & better support in certain enterprise applications
  • In both cases > check if all interactive elements are properly labeled, focusable, and properly ordered
