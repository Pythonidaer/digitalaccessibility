Platform-Specific Tools

iOS

  • VoiceOver and test whether all elements are spoken and operable by swiping.
    • Use iOS Accessibility Inspector to check labels, traits, and contrast.

Desktop

  • Keyboard navigation, screen readers like NVDA or JAWS, or built-in accessibility tools.
  • In both cases > check if all interactive elements are properly labeled, focusable, and properly ordered

Websites

  • Lighthouse (Chrome DevTools: checks for Accessibility, SEO, Performance, etc.)
  • Axe (Chrome Extension: available in Chrome Inspector, with a solid free tier)
  • WAVE (comparable to Axe, but likely has unique features)

PDFs

  • Acrobat Accessibility Checker (Available in Free, and more robust in Pro)

Certifications

  • Earning the following shows you have formal training in accessibility
  • CPACC
    • The exam takes about 2 hours
    • With 100 multiple-choice questions
    • $510 for non-members ($410 for members)
  • WAS
    • Also 2 hours
    • With 75 multiple-choice questions
    • $555 for non-members ($455 for members)
  • Both are from the International Association of Accessibility Professionals (IAAP)
    • Passing both get you the CPWA certification
  • You study independently using recommended materials
    • You then register for an online proctored exam, pass it, then earn the certification

The WCAG's Four Principles

  1. Perceivable: like text alternatives for images
  2. Operable: like keyboard navigation
  3. Understandable: clear language and predictable behavior
    • In this principle, content is clear and predictable
    • Forms should have clear instructions and error messages
    • Navigation should be consistent
    • Language should be readable
    • Users shouldn't be surprised by unexpected behavior
    • Ensure instructions are intuitive, interactions are consistent, and that users can easily understand what to do
  4. Robust: compatible with assistive technologies
  • Each principle breaks down into success criteria like color contrast or form labels
  • I'd be expected to audit content against these criteria
