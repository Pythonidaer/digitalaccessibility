The WCAG's Four Principles

  1. Perceivable: like text alternatives for images
  2. Operable: like keyboard navigation
  3. Understandable: clear language and predictable behavior
    • In this principle, content is clear and predictable
    • Forms should have clear instructions and error messages
    • Navigation should be consistent
    • Language should be readable
    • Users shouldn't be surprised by unexpected behavior
    • Ensure instructions are intuitive, interactions are consistent, and that users can easily understand what to do
  4. Robust: compatible with assistive technologies
  • Each principle breaks down into success criteria like color contrast or form labels
  • I'd be expected to audit content against these criteria

Scope
