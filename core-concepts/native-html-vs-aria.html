Native HTML vs ARIA

Notes from the former Resources page (Miscellaneous: native HTML first) and the former Practice page (“Quick Mental Modal”).

Native HTML first

  • Use native HTML first. Only use ARIA when you have to.
    • Example (bad): <div role="button">Click</div>
    • Example (better): <button>Click</button>

Quick Mental Modal

  • HTML = built-in accessibility
  • ARIA = patch for custom/dynamic UI
